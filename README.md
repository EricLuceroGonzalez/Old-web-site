## 🌟 Descripción del Proyecto

Este repositorio aloja el código fuente del **sitio web personal y portfolio** de **Eric Lucero González**.

El objetivo principal es servir como una plataforma centralizada para mostrar mi experiencia profesional, habilidades técnicas, y una colección de los proyectos personales más relevantes, sirviendo como una tarjeta de presentación digital.

## 🚀 Tecnologías Utilizadas

El sitio está construido principalmente con las siguientes tecnologías:

* **HTML5:** Estructura fundamental del contenido.
* **CSS3 (o Framework si aplica - *ej. Bootstrap/Tailwind*):** Estilización y diseño responsivo.
* **JavaScript (ES6+):** Interactividad, animaciones y manipulación del DOM.
* **GitHub Pages:** Alojamiento y despliegue del sitio.

## 🔗 Acceso al Sitio

El sitio web está desplegado y disponible públicamente a través de GitHub Pages.

## 🛠️ Instalación y Configuración Local

Si deseas clonar el repositorio y ejecutar el sitio localmente, sigue estos sencillos pasos:

1.  **Clonar el Repositorio:**
    ```bash
    git clone [https://github.com/EricLuceroGonzalez/EricLuceroGonzalez.github.io.git](https://github.com/EricLuceroGonzalez/EricLuceroGonzalez.github.io.git)
    ```
2.  **Navegar al Directorio:**
    ```bash
    cd EricLuceroGonzalez.github.io
    ```
3.  **Visualizar Localmente:**
    * Simplemente abre el archivo `index.html` en tu navegador web. No requiere servidor local a menos que se utilicen APIs o funcionalidades de backend.

## 🤝 Contribución

Aunque este es un repositorio de portfolio personal, si encuentras algún error (typo, problema de diseño, etc.), ¡siéntete libre de abrir un *Issue* o enviar un *Pull Request*!

1.  Haz un `fork` del repositorio.
2.  Crea una nueva rama (`git checkout -b feature/nueva-seccion`).
3.  Realiza tus cambios y haz `commit` (`git commit -m 'feat: Añadir nueva sección de contacto'`).
4.  Empuja tus cambios (`git push origin feature/nueva-seccion`).
5.  Abre un `Pull Request`.
